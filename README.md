# Bank Loan Analysis | Power BI + SQL

## Overview
This project analyzes bank loan data to evaluate loan performance, identify risk patterns, and generate business insights using SQL and Power BI.

## Tools & Technologies
- Power BI
- MS SQL Server
- Power Query
- DAX
- Data Modeling

## Objectives
- Analyze loan applications, funded amounts, and repayments
- Identify Good vs Bad loans
- Track trends across time, region, and customer profile
- Support data-driven decision-making in the finance domain

## Key Features
- KPI Cards: Total Applications, Funded Amount, Amount Received, Avg Interest Rate, Avg DTI
- Good Loan vs Bad Loan Analysis
- Monthly Loan Trends
- Regional Analysis by State
- Loan Purpose & Term Analysis
- Home Ownership & Employment Length Insights
- Interactive slicers and navigation

## Data Processing
- Imported and structured data in SQL Server
- Cleaned and transformed data using Power Query
- Created Date Table and relationships
- Built optimized data model

## Business Insights
- Identified high-risk loan segments
- Tracked repayment performance trends
- Highlighted regional loan distribution
- Analyzed customer financial behavior

## Project Outcome
Developed an interactive financial dashboard to monitor loan performance and support risk analysis.
